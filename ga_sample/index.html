GA Sample Teach

Objectives: Students should...

  • Understand programming at a "high level" with control flow
  • Read pseudo code
  • Write pseudo code

Introduce Topic...

  • Begin with the agenda and day's overview (via class website).
  • Explain programming as solving a problem.
  • Introduce pseudo coding as a great tool to begin/setup coding any problem.

Materials/Format: Lecture

Time: 3 minutes

Explain Skill...

  • Use class/lecture example (via Pseudo Coding)
  • Sections: Why, Do, and Don't; Then do example
  • Suggest using a list or table to setup
  • Check understanding after example by asking if anything needs to be repeated

Materials/Format: Code-Along

Time: 7 minutes

Practice Skill Together...

  • Use practice example from Pseudo Coding
  • Encourage students to work through each: Easy, Moderate, Hard
  • Check for understanding by color-coding work and by students asking for help

Materials/Format: Whiteboards

Time: 5 minutes
