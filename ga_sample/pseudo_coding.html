Pseudo Coding

Intro...

Why do it?

  • Pseudo coding helps you think of what you need in your code, the action of your code or how your code runs, and the order or when your code runs.

Do:

  • Identify your code's what, how, and when (at a high level)
  • Follow programming's flow (obey top-down and left-right)

Don't:

  • Get flustered by syntax, naming conventions, or specifics of the code

Example...

Pseudo code to calculate mathematical operations for two numbers:

get num1 get num2 function_add = num1 + num2 function_subtract = num1 - num2 function_multiply = num1 * num2 function_divide = num1 / num2 if addition is selected, then use function_add else if subtraction is selected, then use function_subtract else if multiplication is selected, then use function_multiply else if division is selected, then use function_divide else nothing selected, then ask for inputs
